# 维权文书模板与话术指引

> 本文件包含各类维权场景的文书模板和投诉/举报话术，供生成正式法律文书时参考。
> 模板格式来源于最高人民法院诉讼文书样式、国家信访局、市场监管总局等官方渠道。

---

## 一、消费者投诉书模板

**来源**：市场监管总局《市场监督管理投诉举报处理暂行办法》

```
消费者投诉书

一、投诉人信息
   姓名：XXX        性别：X        联系电话：XXX
   通讯地址：XXX                    邮编：XXX

二、被投诉人信息
   名称（姓名）：XXX
   地址：XXX
   联系电话：XXX（如有）

三、投诉请求
   1. （具体请求：退货、退款、赔偿等）
   2. （如有多项请求，逐条列出）

四、消费者权益争议事实
   （购买/服务的时间、地点、方式、商品/服务信息）
   （争议经过：发现问题→与商家沟通→商家处理态度）
   （权益受损情况及具体损失）

五、法律依据
   （引用相关法律条文，如《消费者权益保护法》第XX条）

六、附件材料
   1. 购物凭证/合同复印件
   2. 商品照片/问题截图
   3. 与商家沟通记录
   4. 其他证据材料

   投诉人（签名）：XXX
   日期：XXXX年XX月XX日
```

## 二、信访申请书模板

**来源**：国家信访局《信访工作条例》

```
信访申请书

信访人：姓名/名称、住址、联系电话、身份证号码
被反映单位/人员：机关/组织/人员名称

信访事项（请求）：
（具体请求、建议、意见，逐条列出）

事实和理由：
（客观真实地陈述事实经过，按时间顺序叙述）
（说明自身权益受到何种侵害）
（说明已采取的救济措施及结果）

此致 XXXX（有权处理的机关名称）

信访人（签名或盖章）：XXX
日期：XXXX年XX月XX日

附件：
1. 相关证据材料
2. 身份证明复印件
3. 其他支持材料
```

## 三、民事起诉状模板

**来源**：最高人民法院诉讼文书样式

```
民事起诉状

原告：XXX，性别X，XXXX年XX月XX日出生，民族X，职业XXX，
      住址：XXX，联系电话：XXX
      （如原告为法人：名称、住所、法定代表人、联系电话）

被告：XXX，性别X，XXXX年XX月XX日出生，民族X，职业XXX，
      住址：XXX，联系电话：XXX
      （如被告为法人：名称、住所、法定代表人、联系电话）

案由：XXXX纠纷（如：买卖合同纠纷、劳动争议、侵权责任纠纷等）

诉讼请求：
1. （具体请求一，如：判令被告退还货款XXX元）
2. （具体请求二，如：判令被告赔偿XXX元）
3. （具体请求三，如：本案诉讼费由被告承担）

事实和理由：
（真实、简明地叙述纠纷经过和结果）
（说明被告的侵权/违约行为）
（说明原告遭受的损失）
（引用相关法律依据）

证据和证据来源，证人姓名和住所：
1. XXX（证据名称），证明XXX，来源：XXX
2. XXX（证据名称），证明XXX，来源：XXX

此致 XXXX人民法院

附：本起诉状副本X份

起诉人（签名或盖章）：XXX
XXXX年XX月XX日
```

## 六、投诉/举报话术指引

### 6.1 消费者投诉话术（12315）

**电话投诉**：
> "您好，我要投诉[商家名称]。我在[时间]在[地点/平台]购买了[商品/服务]，花费[金额]元。发现问题后，[描述问题]。我已与商家协商但[商家态度/处理结果]。我的诉求是[退货/退款/赔偿]。我有[购物凭证/聊天记录/照片]等证据。"

**要点**：
- 说清：谁、什么时间、在哪里、买了什么、花了多少钱
- 说清：问题是什么、造成了什么损失
- 说清：已与商家沟通的结果
- 说清：具体诉求（退款多少、赔偿多少）

### 6.2 劳动维权投诉话术（12333）

> "您好，我要反映劳动权益问题。我在[公司名称]工作，入职时间[XXXX年XX月]，岗位是[XXX]。公司存在[拖欠工资/未签合同/违法辞退/不缴社保]等问题。具体情况是[描述]。我的诉求是[支付工资/双倍工资/赔偿金/补缴社保]。"

### 6.3 报警维权话术

**报案时**：
> "我要报案。在[时间]，在[地点]，[嫌疑人]对我实施了[具体行为]，造成了我[具体损害]。我有[证据描述]。请依法受理并出具受案回执。"

**报案被拒时**：
> "根据《刑事诉讼法》第112条，公安机关对于报案材料应当迅速进行审查，认为有犯罪事实需要追究刑事责任的时候，应当立案。请依法受理并出具受案回执。如不予立案，请出具不予立案通知书。"

**警务督察投诉（12389）**：
> "我要投诉[公安机关名称]。我在[时间]到该单位报案，但[具体不作为情况]。该行为违反了[具体法律规定]。请依法调查处理。"

### 6.4 行政投诉话术（12345）

> "您好，我要投诉[部门名称]。我在[时间]向该部门反映[事项]，但该部门[不作为/乱作为/超时未处理]。具体情况是[描述]。我的诉求是[依法处理/限期答复]。"

### 6.5 退保投诉话术（12378）

> "您好，我要投诉[保险公司名称]。我在[时间]购买了[保险产品]，保费[金额]元。当时销售人员[描述误导行为]。我发现后要求退保，但保险公司[拒绝/拖延/扣费过高]。我的诉求是[全额退保/退还保费]。"

## 七、文书生成注意事项

1. **当事人信息完整**：姓名、性别、出生日期、住址、联系方式必须准确
2. **诉讼请求明确**：每项请求具体、可执行，避免模糊表述
3. **事实简明**：按时间顺序叙述，重点突出，避免冗长
4. **法律依据准确**：引用具体法律条文（条款编号）
5. **证据清单对应**：每项主张都有对应证据支持
6. **格式规范**：使用 A4 页面，标题黑体二号，正文仿宋三号
7. **签名日期**：起诉状/申请书必须手写签名并注明日期
