# Rights Protection Assistant / 维权助手

一个全流程维权辅助 Skill，帮助用户识别侵权行为、整理证据、生成法律文书、规划维权路径，并提供从"意识到被侵权"到"法庭判决/上诉"的完整维权指导。

## 特性

- **11步全流程引导**：从侵权诊断到判决上诉，覆盖维权全链路
- **8大维权类型**：消费者、劳动、租房、金融、行政、报警、互联网/通信、网络纠纷
- **官方权威数据**：所有法律条文、文书模板、渠道信息均来源于政府官网
- **判例查询辅助**：集成裁判文书网、人民法院案例库等权威判例平台
- **多格式文书生成**：支持 Word（起诉状/投诉信）、Excel（证据清单）、PDF（分析报告）
- **5个完整示例案例**：消费者维权、劳动维权、租房维权、报警维权、互联网维权

## 维权全流程

```
Step 1  侵权诊断     → 帮助用户意识到被侵权
Step 2  权益确认     → 确认被侵犯的权益，归类维权类型，匹配法律条文
Step 3  证据整理     → 梳理证据链，明确诉求，预估赔偿金额
Step 4  投诉举报     → 生成投诉信，指引政府职能部门投诉渠道
Step 5  升级举报     → 举报职能部门不作为，生成信访/举报材料
Step 6  调解评估     → 评估行政调解/处罚方案
Step 7  准备起诉     → 生成起诉状/仲裁申请书，提供诉讼流程指引
Step 8  庭审准备     → 生成庭审材料（代理词/答辩状/陈述提纲）
Step 9  判决分析     → 分析判决结果，生成上诉状
Step 10 报警维权     → 报案→受案→立案→不予立案救济→警务督察
Step 11 互联网维权   → APP侵权→个人信息保护→通信服务→工信部/网信办投诉
```

## 法律知识库

知识库涵盖以下法律法规的核心条款和维权要点：

- **《消费者权益保护法》** — 知情权、七天无理由退货、欺诈赔偿（退一赔三）、格式条款
- **《劳动法》《劳动合同法》** — 书面合同、试用期、加班费、经济补偿、违法解除赔偿金、社保
- **《民法典》** — 租赁合同、买卖不破租赁、侵权责任、格式条款、精神损害赔偿
- **《行政复议法》《行政诉讼法》《信访工作条例》** — 行政复议、行政诉讼、信访流程
- **《刑事诉讼法》《公安机关办理刑事案件程序规定》** — 报案受案、立案条件、不予立案救济、警务督察、检察院立案监督
- **《个人信息保护法》《网络安全法》《数据安全法》** — 个人信息权利、APP治理、通信服务申诉
- **《电信条例》《电信用户申诉处理办法》** — 电信用户申诉流程
